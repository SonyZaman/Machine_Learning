# Understanding Data

This module focuses on the initial steps of a machine learning workflow: exploring and understanding your dataset. 

## Contents
- **`01_Understanding_Data.ipynb`**: Jupyter Notebook containing data exploration steps.
- **`train.csv`**: The dataset used for analysis (appears to be the Titanic dataset).

## Key Concepts Covered
1. Loading data using Pandas (`pd.read_csv()`)
2. Inspecting the first few rows (`df.head()`)
3. Checking data types and missing values (`df.info()`)
4. Statistical summary of numerical columns (`df.describe()`)
5. Identifying correlations between features (`df.corr(numeric_only=True)`)

By understanding the data first, we can make informed decisions about data cleaning, feature engineering, and model selection later.
